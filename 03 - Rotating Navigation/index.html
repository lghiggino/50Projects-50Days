A couple of changes

by LNG

This is a really fun project that I decided to tinker. :D Here are a couple of changes I implemented:

  • 1. Now, this is a fully responsive site, meaning: It opens in desktops, notebooks, tablets and cellphones keeping its quality;
  • 2. The links are functional, making this a fully functional website with cool rotating navigation;
  • 3. A fully functional form integrated with formspree - free of charge!
  • 4. After the links on the navigation are clicked, the animation is reset and than the user is forwarded to the new page via JS;

Therefore, to use this site to its full extent, please make sure JavaScript is allowed and is working.
